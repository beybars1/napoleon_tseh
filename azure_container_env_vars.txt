NOTES
================================================================================

1. If DATABASE_URL is set, all POSTGRES_* variables are redundant
2. RABBITMQ_QUEUE is not used by any worker
3. greenapi-worker does NOT need Green API credentials (only saves to DB)
4. ai-agent-worker NEEDS Green API credentials (sends messages to clients)
5. TARGET_CHAT_ID is critical for order processing flow
6. MANAGER_CHAT_IDS routes group chat messages to order processing
